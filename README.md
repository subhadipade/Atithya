# Traditional Cultural & Heritage Showcase Website

A responsive, elegant web application designed to showcase traditional heritage, stunning cultural galleries, immersive itineraries, and timeless aesthetics. Built with a rich visual palette (deep burgundy, gold accents, and warm backgrounds) combined with smooth animations and interactive user elements.

---

## Features

* **Immersive Hero Section:** Full-screen landing area featuring atmospheric background filters and prominent call-to-action buttons.
* **Marquee & Gallery Carousels:** Continuous animated badge bars and image reels showcasing curated heritage highlights.
* **Responsive Mobile Navigation:** A clean, half-screen sliding drawer menu optimized for mobile viewports (`max-width: 768px`).
* **Smooth Scrolling Architecture:** Built-in seamless navigation across all primary page sections.
* **Performance-Optimized CSS:** Structured design utilizing modern CSS variables (`:root`), strict viewport containment (`overflow-x: hidden`), and hardware-accelerated animations.

---

## Project Sections & Structure

The website is organized into the following sequential structural blocks:

* **Home:** `<header class="hero-section">` — Initial landing view with immersive imagery.
* **About:** `<section class="welcome-section scroll-reveal">` — Introduction and background storytelling.
* **Gallery:** `<div class="gallery-whole">` — Visual showcase featuring continuous animation reels.
* **Tour Details:** `<section class="cta-section scroll-reveal">` — Information regarding upcoming tours and cultural experiences.
* **Contact:** `<section class="social-actions-section scroll-reveal">` — Quick access to social handles, mail, and communication buttons.
* **Our Story:** `<footer id="contact" class="footer">` — Footer elements and closing branding details.

---


## Getting Started & Local Setup

1. **Clone or Download** the project repository to your local machine.
2. Ensure your folder contains the required structural layout, including an `Images/` directory for assets like `BG2.jpg`.
3. Open the `index.html` file in any modern web browser to view the application locally.

---

## Tech Stack

* **HTML5:** Semantic markup and modern section structuring.
* **CSS3:** Custom properties, keyframe animations, Flexbox/Grid layouts, and targeted media queries.
* **JavaScript:** Lightweight interactive toggle logic for the mobile navigation drawer.
